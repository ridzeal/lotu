# City/Village/Landmark Level Worldbuilding

This directory contains detailed information about specific locations, settlements, and points of interest where story action takes place.

## Directory Structure

Each location should have its own markdown file: `location_name.md`

## Location File Templates

### City/Village Template

```markdown
# City/Village Name

## Basic Information
- **Continent:** [Parent continent]
- **Type:** [City/Town/Village/Metropolis]
- **Population:** [Number and demographics]
- **Government:** [Leadership type]

## Geography
- Location within continent
- Surrounding terrain
- Climate
- Natural features
- City layout/districts

## Architecture and Infrastructure
- Building styles
- Notable structures
- Transportation
- Utilities
- Defenses

## Inhabitants
- Cultural makeup
- Social classes
- Occupations
- Daily life
- Customs and traditions

## Economy
- Major industries
- Trade goods
- Markets/commerce
- Resources
- Wealth distribution

## Points of Interest
- Temples/religious sites
- Government buildings
- Markets/shops
- Entertainment venues
- Secret locations
- Character homes

## Story Connections
- Important scenes set here
- Character relationships
- Plot developments
- Historical significance
- Hidden secrets

## Connection to Ethereal Being
- Local manifestations
- Legends/stories
- Intervention sites
- Prophetic significance
```

### Landmark/Point of Interest Template

```markdown
# Landmark Name

## Basic Information
- **Type:** [Natural/Artificial/Magical/Tech]
- **Location:** [Within continent/region]
- **Significance:** [Why it matters]

## Description
- Physical appearance
- Size and scale
- Notable features
- Sensory details

## History and Lore
- Origin story
- Historical events
- Cultural significance
- Myths and legends

## Magical/Technological Properties
- Special abilities
- Powers/effects
- Limitations
- Activation methods

## Visitors and Inhabitants
- Guardians/keepers
- Regular visitors
- Dangerous entities
- Population (if any)

## Story Connections
- Plot importance
- Character connections
- Key scenes
- Mysteries to explore
```

## Purpose

Location-level details provide the immediate setting where scenes take place and characters interact. This is the most detailed level of worldbuilding for actual story scenes.
