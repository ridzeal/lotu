# Continent/Ocean Level Worldbuilding

This directory contains information about major landmasses, oceans, and large regions within individual worlds.

## Directory Structure

Each continent/ocean should have its own markdown file: `continent_name.md`

## Continent File Template

```markdown
# Continent Name

## Basic Information
- **World:** [Parent world]
- **Type:** [Continent/Island Chain/Ocean/etc]
- **Size:** [Area in square miles/km]
- **Population:** [Approximate inhabitants]

## Geography and Terrain
- Major mountain ranges
- River systems
- Climate zones
- Natural landmarks
- Bordering oceans/seas

## Nations and Territories
- Countries/states present
- Cultural groups
- Political divisions
- Border conflicts

## Major Cities
Links to specific city files in the `cities/` directory

## Points of Interest
- Natural wonders
- Historical sites
- Magical locations
- Technological installations

## Flora and Fauna
- Unique species
- Dangerous creatures
- Resources
- Environmental challenges

## Connection to Stories
- Which novels feature this continent
- Key plot locations
- Character origins
- Historical events
```

## Purpose

Continent-level details provide the geographical and cultural context for multiple cities and story locations within a region.
