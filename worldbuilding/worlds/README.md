# World Level Worldbuilding

This directory contains information about specific worlds (planets/celestial bodies) where individual novels take place.

## Directory Structure

Each world should have its own markdown file: `world_name.md`

## World File Template

```markdown
# World Name

## Basic Information
- **Universe:** [Parent universe]
- **Type:** [Planet/Moon/Station/etc]
- **Size:** [Comparable to Earth/etc]
- **Population:** [Approximate inhabitants]

## Physical Characteristics
- Terrain overview
- Climate zones
- Natural resources
- Unique geographical features

## Inhabitants
- Dominant species/races
- Cultural groups
- Population distribution
- Language families

## Magic/Technology System
- Power sources
- Magical/tech level
- Limitations and rules
- Integration with daily life

## Political Structure
- Nations/States
- Governance systems
- Conflicts and alliances
- Trade relationships

## Connection to Ethereal Being
- How the being manifests in this world
- Local legends/prophecies
- Historical interventions
- Current influence

## Continents and Major Regions
Links to specific continent files in the `continents/` directory
```

## Purpose

World-level details apply to ALL novels set on that particular world, regardless of time period or specific focus.
