# Universe Level Worldbuilding

This directory contains the highest-level cosmological and metaphysical information about the LOTU multiverse.

## Contents

- **cosmology.md** - Overall multiverse structure and the ethereal being concept
- **multiverse_laws.md** - Physical and metaphysical laws governing the multiverse
- **ethereal_being.md** - Detailed information about the connecting entity across novels

## Purpose

Universal-level worldbuilding applies to ALL novels in the series, establishing the fundamental rules and framework that connect individual stories across different universes.
